= Getting Sonic the Hedgehog Up and Running in Linux

First install the gens emulator ( https://en.wikipedia.org/wiki/Sega_Genesis#Re-releases_and_emulation ). E.g: run `sudo dnf install gens` (or equivalent on non-Fedora).

( I was not successful in getting the “mame” emulator to run the Sonic ROM . As opposed to https://github.com/shlomif/shlomif-tech-diary/blob/master/playing-street-fighter-2-using-mame.asciidoc .)

Download the `.7z` file from
https://archive.org/details/sonic-the-hedgehog-anniversary-edition
and unpack it.

.Type this to run the game.
[source,sh]
----
gens ./\[SHC\]\ \[contest2021\]\ 505\ -\ Sonic\ the\ Hedgehog\ -\ Anniversary\ Edition\ By\ SophieDude.bin
----

Controls: arrow keys, "A", "S", "D", "Enter".
